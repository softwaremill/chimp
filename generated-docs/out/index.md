# chimp: build MCP servers and clients in Scala 3

Chimp is a library for building [MCP](https://modelcontextprotocol.io/specification/2025-03-26) (Model Context Protocol) servers and clients in Scala 3, based on [Tapir](https://tapir.softwaremill.com/). Describe MCP tools with type-safe inputs; talk to MCP servers from your code.
